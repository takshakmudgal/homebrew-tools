# takshakmudgal/tools

Homebrew tap for my terminal utilities.

```sh
brew tap takshakmudgal/tools
```

## Formulas

| tool                              | what it does                                                     |
| --------------------------------- | ---------------------------------------------------------------- |
| `brew install siphon`             | TUI for taking DB dumps + scheduled auto-backups                 |
| `brew install ejector`            | TUI for "what's holding my drive" + ejecting (macOS)             |
| `brew install wa-tui`             | WhatsApp in your terminal                                        |
| `brew install snag`               | IDM-style downloader for videos / files / torrents               |

## Install from the tap

```sh
brew tap takshakmudgal/tools
brew install siphon
brew install ejector
brew install wa-tui
brew install snag
```

## HEAD installs (latest main)

```sh
brew install --HEAD siphon
```

## Updating

```sh
brew update
brew upgrade siphon
```
